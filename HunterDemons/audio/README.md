# Звуки (заготовка)

Клади файлы в `audio/sfx/` и `audio/music/` — форматы **ogg / wav / mp3**.
Имя файла = имя звука в коде, подхватываются автоматически, ничего
настраивать не нужно. Пока файла нет — звук просто не играет.

## audio/sfx/ — эффекты

| Файл | Когда играет |
|---|---|
| `swing` | взмах катаны (каждый удар) |
| `hit` | попадание по демону |
| `crit` | критический удар |
| `skill_sakura` | «Вихрь сакуры» |
| `skill_dash` | «Кибер-рывок» |
| `ult_dragon` | ульта — призыв духа дракона |
| `hero_hurt` | Юкка получает урон |
| `hero_death` | гибель Юкки |
| `heal` | лечение между волнами |
| `demon_attack` | атака демона |
| `demon_death` | смерть демона |
| `demon_spawn` | появление демона |
| `ui_tap` | кнопки меню |
| `ui_next` | перелистывание реплик сюжета |
| `victory` | уровень пройден |
| `defeat` | поражение |

## audio/music/ — музыка (зацикливается автоматически)

| Файл | Где играет |
|---|---|
| `music_menu` | главное меню |
| `music_city` | уровень 1 — Неоновый квартал |
| `music_forest` | уровень 2 — Бамбуковая роща |
| `music_temple` | уровень 3 — Затопленный храм |
